# Paralx

Open source project for transparent dynamic multi-disciplinary finance research.

The project is in early development. The first model page placeholder is
implemented but not yet hosted.

## Structure

```
docs/decisions/   architectural decision records
frontend/         Astro frontend (model pages, components, layouts)
```

## Development

See `CONTRIBUTING.txt` for setup and development instructions.

## Decisions

Architectural decisions are recorded as ADRs in `docs/decisions/`. Each file
covers one decision: what was decided, what alternatives were considered, and
why the chosen option was selected.
