# Contributing

## Prerequisites

**Node.js 22.0.0+**

```bash
node --version
```

**pnpm**

```bash
npm install -g pnpm
```

## Setup

Install dependencies:

```bash
pnpm install
```

## Appendix

Node.js 22.0.0+ specifically is required by Astro 6. See
`docs/decisions/2026-04-25-nodejs-as-frontend-build-runtime.md` and
`docs/decisions/2026-04-18-astro-as-frontend-framework.md`.
